### Key Lessons
1. **PostCSS config**: Tailwind 4 requires an explicit `postcss.config.js` (`plugins: { tailwindcss: {}, autoprefixer: {} }`). Zero-config assumption can break builds silently.
2. **Port integrity before automation**: Verify active ports with `Get-NetTCPConnection -State Listen` before writing Puppeteer scripts that depend on specific ports.
3. **jQuery + React**: Avoid CDN jQuery in Vite/React projects. Install via `npm install jquery` and import as `import $ from 'jquery'`. Mixing CDN and npm creates duplicate globals.
4. **jsPDF in Vite**: Use `npm install jspdf` — do NOT use `<script src="https://cdnjs...">`. UMD vs ESM mismatch causes silent failures.
5. **Puppeteer UAT reliability**: Always `waitForSelector` before `page.click()`. Timeouts without explicit waits cause false negatives.
6. **Hardcoded color elimination**: Migrate hex colors in batches per component, then run a repo-wide grep (`rg '#[0-9a-fA-F]{6}'`) to catch leftovers in one pass.
